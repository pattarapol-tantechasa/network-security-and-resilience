# Network Security & Resilience — Academic Reports

Academic reports from the NSR/AS unit at Swinburne University of Technology (2025).
Topics covered: VPN configuration, public key cryptography, risk analysis, and post-quantum cryptography.

---

## Contents

### Lab 3 — VPNs & OpenVPN
Theoretical overview of VPN characteristics and hands-on configuration of an encrypted 
tunnel between two Ubuntu machines using OpenVPN. Traffic behaviour analysed with 
Wireshark across tunnel (tun1) and ethernet (ens33) interfaces.

**Key concepts:** Tunnelling, IPSec, encryption, Wireshark packet analysis, OpenVPN

---

### Lab 5 — Public Key Cryptography & RSA
Introduction to RSA algorithm with mathematical foundations. Practical exercise 
breaking RSA encryption by recovering a private key from a given public key, 
then decrypting cipher text using MATLAB.

**Key concepts:** RSA key generation, modular arithmetic, encryption/decryption, 
prime factorisation

---

### Case Study — Risk Analysis for a LoRa Underground Mine Network
Structured risk analysis of a LoRa-based emergency communication system for 
underground mining environments. Risks prioritised using the Delphi method. 
Security policies and implementation plans proposed for each identified threat.

**Key concepts:** Delphi method, risk scoring (likelihood × impact), AES-128, 
message authentication, rate limiting, security policy formation

---

### Research Report — Quantum Computing & Cryptography
Research report on the threat quantum computing poses to classical cryptography 
(RSA, ECC) and the transition to post-quantum cryptography (PQC). Covers NIST 
PQC standardisation, lattice-based cryptography, and migration strategies.

**Key concepts:** Shor's algorithm, Grover's algorithm, CRYSTALS-KYBER, 
CRYSTALS-Dilithium, hybrid cryptography, NIST PQC standards, HNDL attacks
